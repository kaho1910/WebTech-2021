Tailwind CSS คืออะไร ?

Tailwind คือ CSS Framework ที่อยู่รูปแบบของ class เช่น flex, border-8, font-bold, items-center และยังมีอื่น ๆ อีกมากมาย

Adam Wathan

ผู้พัฒนา Tailwind CSS

ได้พูดถึงระบบ "semantic class names" ไว้ว่าเป็นสิ่งที่ทำให้ CSS เป็นเรื่องที่ยาก

จากนั้นเลยได้ทำการพัฒนา Tailwind CSS และปล่อย version 1.0 ในปี 2017 และได้รับความสนใจจากบริษัทเทคชั้นนำมากมาย ปัจจุบัน Tailwind CSS เป็น version 2.2 อัพเดตล่าสุดเมื่อเดือนมิถุนายน 2021

ฟีเจอร์โดดเด่นของ Tailwind CSS

CONSTRAINT-BASED

เป็น API สำหรับระบบการออกแบบ

การใช้ชื่อ class สำหรับการตกแต่งแทนการใช้ stylesheets แบบเดิม จะทำให้การจดจ่ออยู่กับงานหลักทำได้ง่ายขึ้น อีกทั้งตัวเลือกการเลือกสี ขนาด เงา และทุก ๆ อย่างยังทำได้โดยง่ายอีกด้วย

BUILD ANYTHING

การใช้งานเหมาะกับงานทุกรูปแบบ

สามารถสร้างได้ไม่ยาก ด้วยส่วนประกอบเดิมที่มีอยู่ก็สามารถเปลี่ยนแนวการแสดงผลได้ทันที

PERFORMANCE

เล็กกระทัดรัด

Tailwind จะทำการนำ CSS ที่ไม่ได้ใช้ออก เมื่ออยู่ในขั้นตอนของการทำ production ทำให้แพกเกจสุดท้ายของ CSS มีขนาดเล็กที่สุด

MOBILE-FIRST

รองรับ Responsive

Tailwind สามารถทำงานกับ responsive design ได้ใน HTML ระบุเพียงแค่ breakpoint ที่ต้องการ

STATE VARIANTS

รองรับสถานะ Hover และ Focus

เพิ่ม hover: ไว้ข้างหน้า class ที่ต้องการให้แสดงผลเมื่อ hover หรือด้วยสถานะอื่น ๆ อย่าง focus, active, disabled, focus-within, focus-visible

COMPONENT-DRIVEN

รองรับการทำงานซ้ำ ๆ ของ components/template

สามารถใช้ซ้ำได้ในทั้ง components หรือ template เพื่อการแสดงผลที่เหมือนกัน นอกจากนี้ยังสามารถใช้ @apply เพื่อระบุ CSS ของ class ใหม่ด้วย class ของ Tailwind

DARK MODE

มีฟังก์ชัน Dark Mode

ใช้ dark: นำข้างหน้า class ที่เกี่ยวกับสี เพื่อให้ทำงานเมื่อมีการเปิดใช้งาน dark mode สามารถเปลี่ยนได้ทั้ง สีพื้นหลัง สีตัวอักษร สีขอบ และอื่น ๆ

CUSTOMIZATION

แต่งรูปแบบได้อย่างอิสระ

Tailwind สามารถเพิ่มรูปแบบการแสดงผลแบบต่าง ๆ ได้ไม่จำกัด

MODERN FEATURES

เทคโนโลยีทันสมัย

Tailwind จะรองรับและอัพเดตฟีเจอร์ใหม่ ๆ และเจ๊ง ๆ ของ CSS อยู่ตลอดเวลา

EDITOR TOOLS

มี extension รองรับ

ไม่ต้องจำชื่อ class เอง เพราะใน VS Code มี extension IntelliSense เป็นตัว autocomplete ให้

READY-MADE COMPONENTS

ทำงานอย่างรวดเร็ว

Tailwind UI เป็นที่ที่สามารถค้นหาตัวอย่างการใช้งานได้อย่างหลากหลาย

วิธีการติดตั้ง

สำหรับวิธีการติดตั้ง Tailwind CSS ก่อนใช้งานในโปรเจคที่เขียนด้วย frameworks หรือเครื่องมือ จะต่างกันออกไปแล้วแต่ที่โปรเจคนั้นใช้

วิธีการติดตั้งสำหรับ frameworks ที่มีใน official documentation ได้แก่

ติดตั้งผ่าน npm

npm install -D tailwindcss@latest postcss@latest autoprefixer@latest

สร้างไฟล์ postcss.config.js

module.exports = {
                    plugins: {
                      tailwindcss: {},
                      autoprefixer: {},
                    }
                  }

สร้างไฟล์การตั้งค่าในแบบของคุณ

npx tailwindcss init

แก้ไฟล์ tailwind.config.js

module.exports = {
                    purge: [],
                    darkMode: false, // or 'media' or 'class'
                    theme: {
                      extend: {},
                    },
                    variants: {},
                    plugins: [],
                  }

เรียกใช้งาน Tailwind ในไฟล์ CSS ของโปรเจคนั้น

@tailwind base;
@tailwind components;
@tailwind utilities;

ใช้งาน Tailwind ใน CLI

ในขั้นตอนนี้ ไฟล์ tailwind.css จะถูกสร้างขึ้น ให้ทำการเรียกใช้งานในไฟล์ HTML เหมือน stylesheets อื่น ๆ ได้เลย

npx tailwindcss -o tailwind.css

ใช้งานไฟล์ CSS แบบ custom โดยใช้ tailwind.css

@tailwind base;
@tailwind components;
@tailwind utilities;

@layer components {
  .btn {
    @apply px-4 py-2 bg-blue-600 text-white rounded;
  }
}

ใช้งาน Tailwind ด้วย CDN

สำหรับวิธีนี้จะได้ฟีเจอร์ไม่ครบ ขาดฟีเจอร์ดังต่อไปนี้

  • การตั้งค่าธีมของ Tailwind
  • ใช้ @apply, @variants เป็นต้น
  • ใช้ group-focus ไม่ได้
  • ไม่สามารถใช้งาน plugin บุคคลที่สาม
  • ฟีเจอร์การลด style ที่ไม่ได้ใช้ทิ้ง
<link href="https://unpkg.com/tailwindcss@^2/dist/tailwind.min.css" rel="stylesheet">

* วิธีจะเหมาะกับการใช้งานพื้นฐาน โปรเจคเล็ก ๆ ที่ไม่ต้องการการ custom
